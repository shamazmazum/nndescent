Common metrics

Package nndescent/point
dist(nil (quote (function (t t) (values (real 0) &optional))))
Similarity metric type
euclidean-dist(x y)

Calculate Euclidean distance for two vectors of single floats.

\(\rho(x, y) = \sqrt{\sum_k (x_k - y_k)^2}\)

manhattan-dist(x y)

Calculate Manhattan distance for two vectors of single floats.

\(\rho(x, y) = \sum_k \lvert x_k - y_k \rvert \)

chebyshev-dist(x y)

Calculate Chebyshev distance for two vectors of single floats.

\(\rho(x, y) = \max_k \lvert x_k - y_k \rvert \)

Random forest

Package nndescent/random-forest
initial-approximation(dist ps k &optional (ntrees 10))
Make an initial approximation for k-NN connectivity graph of a point set ps. An optional parameter ntrees controls the number of random trees used in the process (the bigger value means more accurate result but is slower).

nndescent

Package nndescent/nndescent
nndescent(dist ps approx k &key (max-iterations 5) (min-updates 0))
Return a k-NN connectivity graph of points in the set ps according to the metric dist. An initial approximation of the result approx is required for this algorithm. Parameters max-iterations and min-updates control the termination condition of the algorithm, namely the algorithm terminates if the number of updates of the graph is less than or equal to min-updates.
nndescent!(dist ps approx k &key (max-iterations 5) (min-updates 0))
Inplace version of nndescent.
knn(dist ps queries graph k)
For each point \(q\) in the set queries find k points from the set ps closest to \(q\) using an approximate algorithm called nndescent. graph is l-NN connectivity graph for the points ps, \(l > k\). For \(k = 1 \dots 10\) try \(l = 30 \dots 40\).

Brute-force algorithm

Package nndescent/naive
knn-graph(dist ps k)
Make an exact k-NN connectivity graph of a point set ps. This is an exact and brute-force \(O(n^2)\) algorithm.
knn(dist ps queries k)
For each point \(q\) in the set queries find k points from the set ps closest to \(q\). This is an exact and brute-force algorithm, \(O(nm)\) where \(n\) and \(m\) is a number of points in each set.
